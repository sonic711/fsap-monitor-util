FSAP Monitor Utility

這個 Java 版入口已整合 DuckDB 查詢、最近 SQL 歷史、批次報表產物與監控資料下載，適合離線部署在單一 `jar` 環境中使用。

DB Readonly Schema

Operations

這裡把原本要用 `java -jar ...` 執行的主線命令搬進 UI。主流程固定為 `Doctor -> Ingest -> Sync Views -> Generate Report`，UI 與後端都會一起鎖順序。

目前為 readonly mode，任務按鈕已停用。啟動時可改用 `serve --writable` 或 `--fsap.web.readonly=false`。

依序完成 1 到 4，系統會自動鎖住未到步驟。
